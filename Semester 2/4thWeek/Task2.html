Open Library

Open Library is an online project to create "one web page for every book ever published". Created by Aaron Swartz, Brewster Kahle,[2] Alexis Rossi,[3] Anand Chitipothu,[3] and Rebecca Malamud,[3] This is a project of the Internet Archive.
