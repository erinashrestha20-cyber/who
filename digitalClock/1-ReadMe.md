# ⏰ Digital Clock
A polished digital clock web project with **time-based greetings**, **changing backgrounds**, and a **neon glass UI**.

---

## 📂 Files Overview
- **0-cover.jpg** → Product cover image (branding, ensures authenticity)
- **1-ReadMe.md** → Instructions (this file)
- **2-license.txt** → License terms (Personal, Commercial, Extended)
- **index.html** → Main page for the clock
- **/themes/** → CSS theme variations (Glassmorphism, Neon Glow, Retro Pixel, Minimalist, Modern Rich, Dark Elegant)
- **/Js/** → JavaScript files (`DC.js`, `config.js`)
- **/assets/** → Static image assets (SVGs, favicon, backgrounds)
- **/Screenshots/** → Preview images of included themes

---

## 🚀 Usage
1. Open `index.html` in a browser.  
2. The clock updates every second.  
3. Greeting changes based on the current time of day.  
4. Background uses themed SVG images stored in `assets/`.  

---

## 📝 Customization
- **Username:** Update `CLOCK_CONFIG.username` in `config.js`.  
- **Greetings:** Edit `greeting` values in `config.js`.  
- **Backgrounds/Themes:** Change `bgImage` values in `config.js` or swap CSS in `/themes/`.  
- **Styling:** Adjust fonts, colors, or effects in theme CSS files.  

---

## 🔧 Recommended Edits
- Replace `assets/*.svg` with your own images for custom backgrounds.  
- Adjust CSS in `/themes/` for font size, card color, or button style.  
- Use browser inspector to test CSS changes live.  

---

## ✨ Features
- Live ticking digital clock  
- Manual time correction + reset button  
- AM/PM toggle  
- Dynamic backgrounds (morning, afternoon, evening, night)  
- Personalized greetings with your name  
- Responsive design with neon glass styling
